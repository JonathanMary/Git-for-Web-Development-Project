1. What is Semantic HTML?
    The use of appropriate markup notations in the HTML, to structure the page, help accessibility (ex:for screen readers), and improve SEO.

2. What is HTML used for?
    HTML is used to structure the page.

3. What is an attribute and where do we put it?
    An attribute gives more information about an element. It goes inside the element.

4. What is the h1 tag used for? How many times should I use it on a page?
    The h1 tag is the first header, used as title of the page.

5. Name two tags that have required attributes
    <a>, <img>

6. What do we put in the head of our HTML document? 
    We put the Metadata, info about the page that is not displayed on it.

7. What is an id?
    The id is a unique attribute for a HTML element.

8. What elements can I add an id to? 
    Any HTML element can have an id.

9. How many times can I use the same id on a page? 
    It is a unique attribute, so it is recommended to use it once.

10. What is a class?
    A class is a reusable attribute.

11. What elements can I add a class to? 
    Any element.

12. How many times can I use the same class on a page? 
    As many times as one wants.

13. How do I get my link to open in a new tab?
    By adding the attribute ```target="_blank" to the <a> tag.

14. What is the alt attribute in the image tag used for? 
    Alt is used to describe the image. Used by accessibility and crawlers.

Stretch Questions

1. What is the difference between an inline element and a block element?
    Block starts on new line and takes all the line's width.
    Inline doesn't start on new line, and takes only the width it needs.

2. What happens when an element is positioned absolutely? 
    It is positioned relative to the nearest ancestor.

3. How do I make an element take up only the amount of space it needs but also have the ability to give it a width?
    display: inline-block;

    Bloc: <div>, <section>, <ul>
    Inline: <span>, <button>
    Inline-block: none.
